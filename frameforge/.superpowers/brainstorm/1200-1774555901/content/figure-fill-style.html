How should figures render?

Each figure can be filled, outlined (stroked), or support both independently. Which model fits your design needs?

A

Fill + Stroke (independent)

Each figure has a fill_color + fill_opacity and a stroke_color + stroke_width + stroke_opacity. Either can be set to transparent/zero to use only one. Most flexible — matches how Figma/Illustrator work.

✓ Recommended — covers all cases with one model

B

Fill only (+ optional opacity)

Just a color + opacity. Simpler schema — matches how the existing rectangle and line layer work today. Outlines would need to be zero-width or a separate layer.

C

Stroke only (outline)

Only outline rendering — color, stroke_width_pct, opacity. Clean and simple, but can't do solid filled shapes.
