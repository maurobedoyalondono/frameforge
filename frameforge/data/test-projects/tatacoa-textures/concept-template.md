# Tatacoa Textures — Concept Template

## Series

- **Title:** Tatacoa Textures
- **Project ID:** `tatacoa-textures`
- **Platform:** Instagram Portrait 4:5 — 1080 × 1350 px, 72 DPI, scale_factor 2
- **Frames:** 8
- **Narrative arc:** Landscape → texture → life

---

## Color Palette

| Name | Hex | Role |
|------|-----|------|
| Dusty slate | `#5C6B74` | Overlay color on all text frames |
| Ash white | `#E0D8CE` | All text — eyebrows, headlines, captions |
| Terracotta | `#B85530` | Reserved accent for decorative shapes only — never in text |

---

## Type System

| Role | Family | Usage |
|------|--------|-------|
| Display | `Bebas Neue` | Headlines, location labels |
| Sans-serif | `Source Sans 3` | Eyebrows, captions, all numeric content |

**Size scale (size_pct):**
- Display headline: 7.5–11
- Caption: 2.5
- Eyebrow: 1.8

**Numeral rule:** Source Sans 3 only for numbers. Bebas Neue prohibited for numeric content.

---

## Application Rules

- Overlay color: `#5C6B74` — same on all frames; opacity varies per image
- Text color: `#E0D8CE` — no per-frame changes
- Gradient direction matches text zone: bottom text → `to-bottom`, top text → `to-top`
- No white frames. Full bleed.
- Terracotta `#B85530`: decorative shapes only — available to art director if compositional purpose is clear
